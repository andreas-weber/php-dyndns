php-dyndns
======================

A PHP based DynDNS library

### Prerequisites
- PHP 5.4
- PHP-Curl extension

### Installation

1. Clone repository
```
git clone https://github.com/andreas-weber/php-dyndns.git /path/to/your/folder
```
2. Install dependencies
```
cd /path/to/your/folder
composer update
```
3. Run tests
```
phpunit
```
